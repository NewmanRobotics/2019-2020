# Hardware Manual

## Mappings

```
Motors
Port 0 | DCMotor | Left Wheel
Port 1 | DCMotor | Right Wheel
Port 2 | DCMotor | Arm Lifter
Port 3 | DCMotor | Tape Measure Extender

Servos
Port 1 | CRServo | Cam
Port 2 |  Servo  | Left Foundation Grabber
Port 3 |  Servo  | Left Block Grabber
Port 4 |  Servo  | Right Block Grabber
Port 5 |  Servo  | Right Foundation Grabber

I2C Devices
(bus 0) Port 0 | ModernRoboticsI2cRangeSensor | Distance Sensor
(bus 1) Port 1 |    ModernRoboticsI2cGyro     | Gyro Sensor

```
